## Description
This is an implementation of the NIST defined AES [symmetric-key algorithm](https://en.wikipedia.org/wiki/Symmetric-key_algorithm) written in C++. AES is a variant of the Rijndael [block ciper](https://en.wikipedia.org/wiki/Block_cipher).


This is a class project for **Embry-Riddle Aeronautical University**, class **CS428** (Applied Cryptography).
